# Number Guessing Game

## Description
A Java console-based Number Guessing Game where the player guesses a randomly generated number between 1 and 100.

## Features
- Random number generation
- Hint system
- Limited attempts
- Score calculation
- Replay option

## Technologies Used
- Java

## How to Run
1. Open terminal.
2. Compile the program:
   javac NumberGuessingGame.java

3. Run the program:
   java NumberGuessingGame
